Adult Video Aggregator Project

This website is a streamlined adult video aggregator that organizes and displays content from multiple sources. Videos are listed with thumbnails and titles, linking directly to their original platforms. The user can browse by categories, actors and series, all dynamically sourced from JSON files.

A secure "Protected Mode" unlocks hidden content via a browser-stored password. The site also includes an "Add Video", "Add Actor", and "Add Serie" feature, allowing the user to save new links with metadata. Built with HTML, CSS, and JavaScript, it runs on a server accessible globally through a Grok tunnel. The site uses advanced security measures, including CORS (Cross-Origin Resource Sharing) for safe data access, and incorporates link-cleaning techniques to ensure that only trusted, sanitized links are displayed, providing an extra layer of protection.

This project was a freelance job for a client who authorized me to share it in my portfolio. Due to the nature of the website, all images have been censored, and the files are not publicly shared. However, I can provide access to the live site upon request to showcase my work.

Naval Management Project

This system is designed to optimize cargo transport and route planning for shipping companies. Key features include cargo optimization to maximize profit within weight limits, route planning to find the shortest or most efficient path, and management tools to add, remove, or modify ports and routes.

Built with ANSI C, the system follows a modular and procedural programming approach. It provides a menu-driven interface for users to input cargo details, manage maritime routes, and compute optimal shipping paths using graph-based algorithms and dynamic programming techniques.

Customer Management Project

This project focuses on implementing a client management system for a company that sells computer components. The program allows users to perform various operations on the client database, including viewing, adding, editing, and deleting clients. Additionally, it enables users to search for clients using multiple search algorithms.

Developed in ANSI C, the system follows a modular and procedural programming paradigm. It includes a menu-driven interface for user interactions and supports search algorithms such as sequential search, binary search, ternary search, and interpolation search. The execution time for each search algorithm is measured and displayed to compare their efficiency.

Client data is stored in files, ensuring persistence between program executions. The system is also designed to handle large datasets, allowing tests with different array sizes ranging from 10 to 10,000 clients.

Maritime Travel Planner Project

This is a comprehensive application that enables users to plan and modify sea voyages for both passenger and cargo ships. Key features include planning voyages by selecting ports and calculating trip durations, modifying existing routes, and visualizing the routes through graphs. Users can also save a detailed summary of the trip, including costs and ship details, as a PDF.

The application is built with a user-friendly PyQt6 interface and incorporates various libraries like networkx for graph visualization, fpdf for PDF generation, and tkinter for input dialogs. With this tool, users can efficiently plan, modify, and analyze maritime trips for different ship types.

Cipher Encoding Project

In this project, I explore unique cipher and encoding techniques using Python. It features methods like the Fake First Letter Cipher, Roman-Arabic Cipher, and the Naval Battle Cipher, each with iterative and recursive variations.

Using Python libraries such as Matplotlib, NumPy, and PyQt6, I created an interactive way to encode messages visually and textually.

From simple substitutions to complex recursive encoding, this project showcases creative ways to transform text while experimenting with cryptographic principles.

Numerical Calculus Project

This project explores numerical methods to solve mathematical problems using MATLAB or Octave. It includes Newton-Raphson for fall time estimation, polynomial fitting, interpolation, and numerical integration techniques.

From modeling oxygen concentration based on temperature to calculating fuel mass using integration rules, this project demonstrates the power of numerical analysis in real-world applications.

By leveraging these techniques, I implemented efficient algorithms to tackle mathematical computations with precision.

3D Printing

For the last three years, I’ve been working with 3D printing, starting with two filament 3D printers, Sovol SV02 and Creality Ender 3, using PLA. Later on, I expanded my setup with a resin printer, the HALOT R6 2K, and a Cura cleaning machine specifically for resin prints.

I primarily use Cura and MeshMixer for the 3D models and slicing in the filament printers. For resin printing, I use Lychee Slicer for the slicing process.

I have 3D printed a variety of items, from decorative objects and household items to cosplay accessories. With the resin printer, I have made detailed figures and tabletop models, which I painted after printing. Additionally, I’ve sold some of the items I’ve printed and painted.
